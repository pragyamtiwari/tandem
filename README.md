# Tandem

**Project Name:** Tandem

**Project Pitch:** Tandem is an online collaborative code editor that allows multiple users to concurrently edit code in a synchronized manner. To encourage collaboration, it includes in-built voice and video call functionality and the ability to run and execute code snippets.

**Project Description:** This project simplifies the online interview process. While other collaborative code editors exist, few have in-built communication features. For code execution, Docker containers will ensure safe and secure snippet execution without affecting the application. Web-sockets enable live, concurrent text file changes. The project supports syntax highlighting and multiple popular programming languages.

**Primary Users:** Software engineers and recruiters

**Features:**
- Real-time code editing with live cursors
- Built-in voice and video calls
- Run code safely with Docker
- Syntax highlighting for multiple languages
- Share sessions with room links

**Benefits:**
- No need to switch between multiple tools
- Faster interview setup
- Test code instantly
- Nature conversation while coding
